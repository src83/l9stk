## l9stk - Laravel 9 StarterKit

- Version: v9.52.21
- PHP: 8.2.30

#### Install
- `git clone ...`
- append and confirurate `.env`
- `composer update`
- `php artisan migrate`
- `php artisan db:seed`
- `php artisan ide-helper:models`
- `npm install`
- `npm run dev`
- open in browser `http://l9stk.loc`
